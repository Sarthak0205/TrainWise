# TrainWise

Train Smarter. Progress Consistently.

---

## Description

TrainWise is a workout tracking, progress analytics, recovery monitoring, and intelligent training guidance platform.

It helps athletes understand their performance, track personal records, monitor recovery, and make smarter training decisions over time.

---

## Key Features

- **Workout Management**: Log training sessions, explore exercise history, ingest data from standard CSV logs, and manage athlete profiles.
- **Progress Tracking & Analytics**: Visualize weekly training volume changes, pattern balance ratios, and historical progression logs.
- **Recovery Monitoring**: Evaluate systemic fatigue coefficients and dynamically adjust overload targets to prevent plateaus.
- **Intelligent Training Guidance**: Access personalized weight, rep, and progression suggestions based on current performance states.
- **Personal Records (PR) Hall of Fame**: Log and celebrate compound lift milestones and maximum estimated 1RM achievements.

---

## Technology Stack

- **Frontend**: React + Vite, Tailwind CSS / Vanilla CSS, Lucide icons
- **Backend API**: Node.js + Express
- **Database**: MongoDB
- **Model Engine**: FastAPI + Random Forest Models (Machine Learning)
- **Security & User Management**: JWT Authentication, secure hashing, and custom routes
